# recitalk 개인정보처리방침

GGjs(이하 "운영자")는 모바일 애플리케이션 **recitalk**(이하 "서비스")를 운영함에 있어 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수하기 위하여 다음과 같이 개인정보처리방침을 수립·공개합니다.

본 개인정보처리방침은 2026년 6월 3일부터 적용됩니다.

---

## 1. 개인정보의 처리 목적

운영자는 다음의 목적을 위하여 개인정보를 처리합니다. 처리한 개인정보는 아래 목적 이외의 용도로 이용하지 않으며, 이용 목적이 변경되는 경우 관련 법령에 따라 필요한 조치를 이행합니다.

| 처리 목적 | 상세 내용 |
|---|---|
| 회원가입 및 계정 관리 | 회원 식별, 로그인, 회원탈퇴 처리, 서비스 이용자 관리 |
| 맞춤형 레시피 추천 | 이용자의 나이, 성별, 요리 레벨, 보유 재료 및 서비스 이용 행동을 기반으로 개인화된 레시피 추천 제공 |
| 보유 재료 관리 | 이용자가 직접 입력하거나 OCR 기능을 통해 등록한 재료 정보 관리 |
| 즐겨찾기 기능 제공 | 이용자가 저장한 레시피 관리 |
| AI 챗봇 기능 제공 | 이용자의 질문 및 서비스에서 추천된 레시피 정보를 기반으로 요리 관련 답변 제공 |
| OCR 기능 제공 | 이용자가 촬영하거나 업로드한 영수증 또는 재료 사진에서 식재료 정보를 추출 |
| 서비스 품질 개선 | 레시피 클릭, 저장, 조리 완료, 별점 정보 등을 활용한 추천 정확도 개선 |
| 문의 및 권리 행사 처리 | 개인정보 관련 문의, 계정 및 데이터 삭제 요청 처리 |

## 2. 처리하는 개인정보의 항목 및 수집 방법

운영자는 서비스 제공을 위하여 다음과 같은 개인정보를 처리합니다.

### 2.1 회원가입 및 프로필 정보

| 항목 | 수집 방법 | 이용 목적 |
|---|---|---|
| 아이디 | 회원가입 시 이용자 입력 | 회원 식별 및 로그인 |
| 비밀번호 | 회원가입 시 이용자 입력 | 로그인 인증 |
| 이메일 주소 | 회원가입 시 이용자 입력 | 계정 관리 및 문의 대응 |
| 나이 | 회원가입 또는 프로필 설정 시 이용자 입력 | 개인화 추천 제공 |
| 성별 | 회원가입 또는 프로필 설정 시 이용자 입력 | 개인화 추천 제공 |
| 요리 레벨 | 회원가입 또는 프로필 설정 시 이용자 입력 | 개인화 추천 제공 |

비밀번호는 원문 그대로 저장하지 않고, 단방향 해시 처리된 형태로 저장됩니다.

### 2.2 서비스 이용 과정에서 생성·처리되는 정보

| 항목 | 수집 방법 | 이용 목적 |
|---|---|---|
| 보유 재료 정보 | 이용자 직접 입력 또는 OCR 추출 결과 등록 | 보유 재료 기반 레시피 추천 |
| 즐겨찾기한 레시피 정보 | 이용자의 저장 기능 이용 | 즐겨찾기 기능 제공 |
| 레시피 클릭 기록 | 이용자의 서비스 이용 과정에서 생성 | 개인화 추천 및 추천 정확도 개선 |
| 레시피 저장 기록 | 이용자의 서비스 이용 과정에서 생성 | 개인화 추천 및 추천 정확도 개선 |
| 조리 완료 기록 | 이용자의 서비스 이용 과정에서 생성 | 개인화 추천 및 추천 정확도 개선 |
| 별점 정보 | 이용자의 서비스 이용 과정에서 생성 | 개인화 추천 및 추천 정확도 개선 |
| Redis 캐시 데이터 | 서비스 이용 과정에서 임시 생성 | LLM 응답 캐싱 및 중복 피드백 방지 |

### 2.3 AI 챗봇 기능 이용 시 처리되는 정보

| 항목 | 처리 방법 | 이용 목적 |
|---|---|---|
| 이용자가 입력한 질문 또는 대화 내용 | 이용자가 AI 챗봇 기능을 이용할 때 처리 | 요리 관련 답변 생성 |
| 서비스에서 추천한 레시피 정보 | 이용자의 질문과 함께 AI 서비스에 전송 | 추천 레시피에 관한 맞춤형 답변 생성 |

AI 챗봇 대화 내용은 대화 세션이 유지되는 동안 이용자의 기기 또는 앱 화면 내에서 임시로 유지될 수 있으나, 운영자의 MongoDB 데이터베이스에는 저장되지 않습니다.

다만, 서비스 제공 및 중복 피드백 방지를 위하여 사용자 ID와 연결된 일부 AI 응답 캐시 또는 요청 식별값이 Redis에 최대 1시간 동안 임시 저장될 수 있습니다.

AI 답변 생성을 위해 외부 AI 서비스 제공자에게 전송되는 정보의 처리에 관한 사항은 본 방침의 "개인정보의 국외 이전에 관한 사항"을 확인하여 주시기 바랍니다.

### 2.4 OCR 기능 이용 시 처리되는 정보

| 항목 | 처리 방법 | 이용 목적 |
|---|---|---|
| 이용자가 촬영하거나 업로드한 영수증 또는 재료 사진 원본 | OCR 기능 이용 시 외부 AI 서비스에 전송 | 영수증 또는 사진 속 식재료 정보 추출 |

OCR 기능을 위해 전송된 영수증 또는 재료 사진 원본은 운영자의 데이터베이스에 저장되지 않으며, 식재료 추출 처리가 완료된 후 운영자가 별도로 보관하지 않습니다.

다만, 이용자가 추출된 식재료 정보를 자신의 보유 재료로 등록하는 경우, 등록된 보유 재료 정보는 서비스 제공을 위하여 저장됩니다.

이용자는 OCR 기능 이용 시 사진에 불필요한 개인정보가 포함되지 않도록 주의하여 주시기 바랍니다.

## 3. 개인정보의 처리 및 보유기간

운영자는 개인정보의 처리 목적이 달성되거나 이용자가 회원탈퇴를 하는 경우 해당 개인정보를 지체 없이 삭제합니다.

| 구분 | 보유기간 |
|---|---|
| 회원가입 및 프로필 정보 | 회원탈퇴 시까지 |
| 보유 재료 정보 | 회원탈퇴 시까지 |
| 즐겨찾기 정보 | 회원탈퇴 시까지 |
| 레시피 클릭, 저장, 조리 완료, 별점 정보 | 회원탈퇴 시까지 |
| Redis 캐시 데이터 | 저장 후 1시간 이내 자동 삭제 |
| AI 챗봇 대화 내용 | 운영자 데이터베이스에 저장하지 않으며, 앱 내 세션 종료 시 임시 유지 정보 삭제. 단, 서비스 제공을 위해 일부 AI 응답 캐시 또는 요청 식별값이 Redis에 최대 1시간 동안 임시 저장될 수 있음 |
| OCR 영수증 또는 재료 사진 원본 | 운영자 데이터베이스에 저장하지 않으며, 추출 처리 완료 후 운영자가 보관하지 않음 |
| 서버 로그 | Render 서비스 운영 과정에서 생성될 수 있으며, Render 플랜에 따른 로그 보관기간 경과 후 삭제 |
| 개인정보 관련 문의 내용 | 문의 처리 완료 후 지체 없이 삭제 |

운영자는 별도의 법령상 보관 의무가 없는 한, 회원탈퇴 이후 이용자의 개인정보를 별도로 보관하지 않습니다. 이용자가 탈퇴 후 재가입하는 경우 기존 정보는 복구되지 않으며 신규 이용자로 처리됩니다.

단, Anthropic 및 OpenAI API를 통해 처리되는 정보는 각 외부 서비스 제공자의 보안 및 악용 방지 목적에 따라 일정 기간 처리될 수 있으며, 자세한 내용은 본 방침의 "개인정보의 국외 이전에 관한 사항"을 확인하여 주시기 바랍니다.

## 4. 만 14세 미만 아동의 개인정보 처리에 관한 사항

서비스는 만 14세 미만 아동을 대상으로 하지 않으며, 만 14세 미만 이용자의 회원가입 및 서비스 이용을 허용하지 않습니다.

운영자는 만 14세 미만 아동의 개인정보가 수집된 사실을 확인한 경우 해당 정보와 계정을 지체 없이 삭제합니다.

---

## 5. 개인정보의 제3자 제공에 관한 사항

운영자는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다.

다만, 다음의 경우에는 관련 법령에 따라 개인정보를 제공할 수 있습니다.

1. 이용자가 사전에 동의한 경우
2. 법령에 특별한 규정이 있거나 법령상 의무를 준수하기 위하여 필요한 경우
3. 수사기관 또는 감독기관이 적법한 절차에 따라 요청하는 경우

서비스 제공을 위하여 외부 업체가 개인정보를 처리하는 경우에는 아래의 개인정보 처리업무 위탁 및 국외 이전에 관한 사항에 따라 처리합니다.

## 6. 개인정보 처리업무의 위탁에 관한 사항

운영자는 원활한 서비스 제공을 위하여 다음과 같이 개인정보 처리업무를 위탁하거나 외부 서비스를 이용하고 있습니다.

| 수탁업체 또는 서비스 | 위탁업무 내용 | 처리되는 정보 | 보관 또는 처리 위치 | 보유기간 |
|---|---|---|---|---|
| MongoDB, Inc. (MongoDB Atlas) | 서비스 데이터베이스 운영 및 저장 | 회원가입 정보, 프로필 정보, 보유 재료, 즐겨찾기, 레시피 클릭·저장·조리 완료·별점 정보 | AWS Seoul Region, 대한민국 | 회원탈퇴 시까지 |
| Render Services, Inc. (Render) | 백엔드 서버 호스팅 및 서비스 운영 | 서비스 요청 처리 과정에서 필요한 계정 식별 정보, 서비스 이용 요청 정보, 서버 로그 및 Redis 캐시 데이터 | Virginia, United States | 서비스 제공 기간 동안 처리되며, Redis 캐시 데이터는 저장 후 1시간 이내 자동 삭제 |
| Anthropic PBC | Claude API를 이용한 AI 챗봇 답변 생성 | 이용자가 입력한 질문 또는 대화 내용, 서비스에서 추천한 레시피 정보 | 미국 | API 제공자의 정책에 따름 |
| OpenAI OpCo, LLC | OpenAI API를 이용한 이미지 기반 식재료 추출 | 이용자가 촬영하거나 업로드한 영수증 또는 재료 사진 원본 | 미국 | API 제공자의 정책에 따름 |

Redis는 Render 배포 환경에서 운영되며, LLM 응답 캐싱 및 중복 피드백 방지 목적으로 사용됩니다. Redis에는 사용자 ID와 AI 응답 캐시 또는 요청 식별값 등 서비스 제공에 필요한 정보가 최대 1시간 동안 임시 저장될 수 있으며, 해당 정보는 저장 후 1시간 이내 자동 삭제됩니다.

운영자는 위탁계약 또는 외부 서비스 이용 과정에서 개인정보가 위탁 목적 외로 처리되지 않도록 필요한 조치를 취하고, 수탁업체 또는 서비스가 변경되는 경우 본 개인정보처리방침을 통해 공개합니다.

## 7. 개인정보의 국외 이전에 관한 사항

서비스는 백엔드 서버 운영, AI 챗봇 및 OCR 기능 제공을 위하여 이용자의 개인정보를 국외에 위치한 외부 서비스 제공자에게 전송하거나 국외에서 처리할 수 있습니다.

이용자는 AI 챗봇 또는 OCR 기능을 사용하지 않는 방법으로 해당 기능과 관련된 국외 이전을 거부할 수 있습니다. 다만, 국외 이전을 거부하는 경우 AI 챗봇 기능 또는 OCR 기능의 이용이 제한됩니다.
